## ADDED Requirements

### Requirement: Toolbar tool buttons are displayed at a larger size
The application SHALL render editor toolbar buttons with larger icons and larger label text than the current default, improving readability and click target size on all screen sizes.

#### Scenario: Tool buttons are visually larger than before
- **WHEN** the application is running with a document open
- **THEN** the toolbar tool buttons are noticeably larger in both icon size and label text

#### Scenario: All tool buttons use the consistent larger size
- **WHEN** the toolbar is displayed
- **THEN** all tool buttons (Select, Text, Textarea, Checkbox, Image) are the same larger size
